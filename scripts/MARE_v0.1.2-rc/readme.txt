Getting started with MARE:
--------------------------

1. open a terminal.
2. change to the directory with the source code and the
   makefile in it.
3. Enter:  
           make

4. Now you can move the binary 'MARE' to any other direc-
   tory.
5. To run MARE with default settings, move the concate-
   nated alignment and the charset file to the same di-
   rectory as 'MARE'. Then enter:
   
          ./MARE yourfile.charset yourfile.fas

5. For further informations enter: 
 
          ./MARE -h
